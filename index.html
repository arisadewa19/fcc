Cara Install di HP

iPhone (Safari):
Buka website ini di Safari → tap ikon Share (kotak dengan panah) → pilih "Add to Home Screen" → tap "Add"

Android (Chrome):
Buka website ini di Chrome → tap menu titik tiga (⋮) → pilih "Add to Home Screen" atau akan muncul banner install otomatis
