Panduan Setup Google Sheets (15 menit)
1
Buat Google Sheet baru
Buka sheets.google.com → klik "+" buat sheet baru → beri nama "Financial Command Center"
2
Buka Apps Script
Di Google Sheet, klik menu Extensions → Apps Script
3
Paste kode script
Hapus semua kode yang ada, lalu paste kode dari file Code.gs yang sudah saya berikan. Klik Save (ikon disket).
4
Deploy sebagai Web App
Klik Deploy → New deployment → pilih type Web app
• Execute as: Me
• Who has access: Anyone
Klik Deploy → izinkan akses → salin URL yang muncul
5
Paste URL di atas
Paste URL dari langkah 4 ke kolom URL di atas → klik "Simpan & Test Koneksi"
Jika muncul ✅ — selesai! Data Anda sekarang tersimpan di Google Sheets.
