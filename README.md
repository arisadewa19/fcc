# Financial Command Center — Panduan Setup

## Cara Deploy ke GitHub Pages (GRATIS, 10 menit)

### Langkah 1: Buat akun GitHub
Buka https://github.com dan daftar gratis jika belum punya.

### Langkah 2: Buat repository baru
1. Klik tombol hijau "New" di halaman utama GitHub
2. Nama repository: `fcc` (atau nama lain yang Anda suka)
3. Pilih "Public"
4. Klik "Create repository"

### Langkah 3: Upload file
1. Di halaman repository, klik "uploading an existing file"
2. Upload ketiga file ini:
   - `index.html`
   - `manifest.json`
   - `sw.js`
3. Klik "Commit changes"

### Langkah 4: Aktifkan GitHub Pages
1. Klik tab "Settings" di repository
2. Scroll ke bawah, klik "Pages" di menu kiri
3. Di bagian "Source", pilih "Deploy from a branch"
4. Branch: pilih "main" → folder "/ (root)"
5. Klik "Save"

### Langkah 5: Akses dashboard
Setelah 1-2 menit, dashboard Anda aktif di:
**https://[username-github-anda].github.io/fcc**

Simpan link ini di WhatsApp catatan sendiri!

## Cara Install di HP

### iPhone:
1. Buka link di Safari
2. Tap ikon Share (kotak + panah ke atas)
3. Pilih "Add to Home Screen"
4. Tap "Add"
→ Ikon "Finance" muncul di layar utama

### Android:
1. Buka link di Chrome
2. Akan muncul banner "Install app" otomatis
3. Tap "Install"
→ Ikon muncul di layar utama

---

## Cara Update Data dari HP

1. Buka app Finance dari layar utama
2. Tap ikon ⚙️ (Atur) di menu bawah
3. Update nilai portofolio
4. Tap "Simpan & Update"

Untuk catat transaksi harian:
1. Tap ✏️ (Tracking)
2. Pilih jenis, kategori, jumlah
3. Tap "+ Catat Transaksi"

---

## Pertanyaan?
Data tersimpan di browser HP Anda (localStorage).
Untuk backup, buka Pengaturan → Backup Data → Salin backup.
